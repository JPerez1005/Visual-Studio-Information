Mas Información

Windows

Visual Studio es la elección natural para desarrollar aplicaciones para la plataforma Windows, ya sean aplicaciones de escritorio (Windows Forms, WPF), aplicaciones web (ASP.NET), aplicaciones móviles (Xamarin), o juegos (Unity).

Apple

Visual Studio for Mac permite desarrollar aplicaciones multiplataforma, incluyendo aplicaciones móviles con Xamarin, aplicaciones web con ASP.NET Core y aplicaciones de escritorio.

Linux

Visual Studio Code es un editor de código abierto, altamente personalizable y ligero que se ejecuta en Linux, Windows y macOS.

Integración con Xamarin

Facilita el desarrollo de aplicaciones móviles para iOS y Android utilizando Xamarin, con acceso a herramientas de diseño y pruebas en dispositivos reales y simuladores.
